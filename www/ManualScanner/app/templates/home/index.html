Welcome to scan.bugbounty.site

scan.bugbounty.site is an online web application recon tool free for use for all hackers. Through this application, you can find information regarding your target and also be able to find some basic vulnerabilities.

We are still enhancing the application with more features so currently, this is only open for selected hackers. If you want to be a beta tester of the application and give feedback, please email bugbountysite@gmail.com.

Specification:

Currently scan.bugbounty.site does multiple types of scan against a target. The design behind this application is to make sure researchers can use this to do 90%+ of their reconnaissance against a target.

While it is still on construction mode, scanner currently supports following things:

  • DNS Scan (A, MX, NS).
  • Subdomain scanning with auto takeovers for some services.
  • Report generation upon successful subdomain takeover.
  • Censys assets search
  • S3 Bucket scanner
  • Publicly showcased vulnerabilities
  • Bruteforce directories for root domain

Planned future update will include additional features:

  • Search keywords for dorkings/github recon.
  • Mini Directory Bruteforce for each subdomains.
